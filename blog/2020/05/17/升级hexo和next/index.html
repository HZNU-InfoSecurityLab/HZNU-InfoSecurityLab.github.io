当我这么做时，却可耻地失败了。原来我当初是通过 cURL 而非 git clone 的方式做的安装。无奈，这次没法享受 git merge 的便利了。至于原先的配置文件、插件包和任何做过的代码改动，我只能手动复制和比对。

为了今后更新的便利，我决定通过 git clone 重新安装 NexT 。
